EASY

About:

We developed some basic Virtual Machine Images allowing you to build application using Intel's trusted execution environment intel sgx. These images provide all the sessasary tools in order to have as painless development using the intel SGX platform including sgx_driver, SDK and PSW, packaged in an Ubuntu 16.04 LTS GNU/Linux distro.
