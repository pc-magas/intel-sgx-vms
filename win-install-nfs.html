Install NFS into MS-Windows

For MS windows based systems is recomended to install the NFS server via Cygwin.
In order to do that follow theese steps:
  1. Run the cygwin installer as Administrator
  2. Press next Until you see the following window:
  3. Select "Full" from the dropdown menu. Afterwards search in the search bar as seen in the image bellow:
  4. Then search and select the following:
    • nano
    • nfs-server
    • cygrunsrv
    In order to install it just click on the entry.
  5. In order to perform the installation click on next button.
  6. When download has finished press Finish button
